# Changelog

## v1.3.0 — 2026-06-12

### Added
- 账单详情：点击账单条目查看完整信息（金额、设备、交易号等）
- 设备移除：长按设备卡片可取消收藏移除设备
- 积分余额实时显示：「我的」页面自动加载积分数据
- 积分抵扣开关：消费时可选择是否使用积分抵扣

## v1.2.0 — 2026-06-12

### Added
- 积分中心：任务列表赚积分、积分明细历史
- 消费记录：饮水账单列表，支持查看金额和时间
- 积分 MD5 签名算法实现，支持一键领取积分
- 「我的」页面新增积分中心、消费记录入口

## v1.1.3 — 2026-05-20

### Changed
- 扫码添加设备改为输入设备码绑定，更稳定可靠

## v1.1.2 — 2026-05-20

### Changed
- 移除底部导航扫码标签，功能合并至「我的」页面
- 个人信息与积分卡片从首页移至「我的」页面
- 首页精简为纯设备列表

## v1.1.1 — 2026-05-20

### Fixed
- 修复登录成功后路由跳转错误导致显示"登录失败"的问题

## v1.1.0 — 2026-05-20

### Added
- 底部导航栏三大标签（首页 / 扫码 / 我的）
- 「我的」页面：用户信息、主题切换、仓库地址
- 深色模式切换，主题自动持久化

### Changed
- 扫码添加设备和退出登录移至「我的」页面
- 首页 AppBar 精简，操作入口统一管理

## v1.0.0 — 2026-05-19

### Added
- 手机号 + 图片验证码 + 短信验证码三步登录
- 首页设备列表，展示账户信息与积分
- 扫码绑定饮水机设备
- 设备详情页：累计出水量、本次接水量、余额
- 一键取水 / 停止取水（确认弹窗防误触）
- 计费方式选择（按量计费 type=21 / 按次计费 type=91）
- 取水状态实时轮询（3 秒间隔）
- 手动关水自动检测，UI 即时更新
- Token 持久化，退出后自动登录
- 设备列表离线缓存，二次打开秒出
